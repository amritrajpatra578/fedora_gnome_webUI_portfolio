# Fedora GNOME Web UI Portfolio

A system-inspired portfolio where I recreate the **Fedora GNOME desktop experience in the browser**, focusing on real UI behavior, precision, and interaction design.

Instead of building a typical portfolio, I wanted to build something that feels like a **real system**, where interactions, layout, and small details matter.

## Features

### GNOME-style Top Bar

- Centered real-time clock
- System indicators (Wi-Fi, Bluetooth, Battery, Volume)
- Blur and transparency effects

### Quick Settings Panel

- Interactive toggles
- Volume & brightness sliders
- Floating panel with layered UI feel

### Clock Menu

- Functional calendar with month navigation
- Correct weekday alignment logic
- Current date highlighting
- “Now Playing” section

### Mini Media Player

- Album cover + track info
- Play / Pause controls
- Minimal GNOME-inspired layout

### Dock (Bottom Bar)

- Centered app icons
- Clean and responsive layout
- Inspired by GNOME/macOS behavior

Most portfolios focus on CRUD apps or dashboards.  
I wanted to explore something different — **system-level UI designed portfolio**.
